# 🌟 Teacher Sanchez Plataforma

Plataforma interactiva para la enseñanza del inglés con juegos educativos.

## 📋 Descripción

Esta es una aplicación web interactiva diseñada para enseñar inglés de forma lúdica y divertida mediante trivia de colores y otras actividades educativas.

## ✨ Características

- 🎮 **Trivia Interactiva**: Preguntas sobre colores en inglés
- 🎤 **Síntesis de Voz**: Pronunciación automática de las preguntas
- 🎨 **Interfaz Colorida**: Diseño atractivo y amigable para estudiantes
- ⭐ **Sistema de Puntuación**: Registro de progreso del estudiante
- 🏆 **Pantalla de Victoria**: Celebración al completar todas las preguntas
- 📱 **Responsive**: Compatible con dispositivos móviles

## 🚀 Uso

1. Abre el archivo `index.html` en tu navegador
2. Lee o escucha la pregunta
3. Selecciona la respuesta correcta
4. Avanza a la siguiente pregunta
5. ¡Completa todas las preguntas para ganar!
